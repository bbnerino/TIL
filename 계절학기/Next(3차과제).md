# Create Next App + React + TypeScript + next.js 로 프론트엔드 프로젝트 토대 쌓기

### 왜 React인가

- react 시장이 국내에서는 압도적이다.
- SSR (Server Side Rendering) 이 필요해졌다
  - CSR은 검색엔진을 통한 노출이 약하다

### TypeScript

**JavaScript**를 기반으로 정적 타입 문법을 추가한 언어

→ runtime에서만 오류를 발견 가능한데

TypeScript는 코드 작성 단계에서 오류를 확인할 수 있다.

## 환경구축

1. 쉘 다운로드 ( git bash)

2. Node.js 설치

3. Next.js 설치

   - `$ yarn create next-app --typescript`

   - 서버실행 `$ npm run dev`

     **Welcome to Next.js!**
